# AgenticAI Profile Website

Professional personal portfolio built with Next.js, React, Tailwind CSS, and TypeScript.

## Setup

1. Install dependencies

```bash
npm install
```

2. Run locally

```bash
npm run dev
```

3. Open http://localhost:3000

## Deployment

### Vercel

1. Create a Vercel account at https://vercel.com
2. Import this repository
3. Use the default Next.js settings
4. Deploy

### Netlify

1. Create a Netlify account at https://netlify.com
2. Add a new site from Git
3. Connect your repository
4. Set build command to `npm run build`
5. Set publish directory to `.next`
6. Deploy

## Customize your profile

- Replace `public/profile.svg` with your own image file as `public/profile.jpg` or `public/profile.png`.
- Update contact information in `app/page.tsx`.
- Add more projects or experience sections.

- Demo: Deployed to production URL: https://tahsins-profile.netlify.app
